Summary: Latitude vs. X

The purpose of this project was to compare weather at different latitudes across the globe at a specific point in time. The Open Weather API from OpenWeather.com was used to pull data from approximately 600 random cities across the globe. This data was then cleaned and plotted using Python Pandas and Matplotlib.
